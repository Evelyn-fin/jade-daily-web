# Jade's Daily Planner - 测试指南

## 🌐 当前访问方式

**注意：** 当前应用运行在开发环境，仅在沙箱内部可访问。你需要通过以下方式测试：

### 方式1：本地开发（推荐用于调试）

在你的本地电脑上克隆或访问代码仓库后运行：

```bash
# 1. 进入项目目录
cd /workspace/projects

# 2. 安装依赖（如果未安装）
pnpm install

# 3. 启动服务
coze dev
```

然后访问：
- **前端地址：** http://localhost:5000
- **后端API：** http://localhost:9091/api/v1

### 方式2：微信/飞书访问（需要配置内网穿透）

如果你想在微信或飞书中使用，需要：

1. **前端部署到云服务**（如 Vercel、Netlify、腾讯云等）
2. **配置内网穿透**（如 frp、ngrok），让前端能访问本地后端

## 🔄 API 测试示例

使用 curl 或 Postman 测试后端接口：

```bash
# 1. 获取今日语录
curl http://localhost:9091/api/v1/daily-quotes/today

# 2. 获取今日待办清单
curl http://localhost:9091/api/v1/todos

# 3. 创建新待办
curl -X POST -H "Content-Type: application/json" \
  -d '{"title":"测试新待办","category":"日常"}' \
  http://localhost:9091/api/v1/todos

# 4. 获取日历统计（2026年3月）
curl "http://localhost:9091/api/v1/todos/calendar?month=2026-03"

# 5. 获取指定日期的待办清单
curl http://localhost:9091/api/v1/todos/date/2026-03-18

# 6. 创建灵感（AI自动生成待办）
curl -X POST -H "Content-Type: application/json" \
  -d '{"content":"我想在一个月内学会游泳"}' \
  http://localhost:9091/api/v1/inspirations

# 7. 获取所有灵感记录
curl http://localhost:9091/api/v1/inspirations
```

## ⚠️ 注意事项

1. **AI生成需要时间**：创建灵感后，AI生成待办事项需要 10-15 秒，请耐心等待
2. **数据本地存储**：所有数据存储在 SQLite 数据库中（`server/data/jade-daily.db`）
3. **刷新页面**：查看 AI 生成的待办事项时，可能需要刷新页面
4. **后端服务**：确保后端服务正常运行（端口 9091）

## 🐛 常见问题

**Q: 为什么 AI 没有生成待办事项？**
A: AI 生成需要时间，请等待 10-15 秒后刷新页面查看。

**Q: 如何清空所有数据？**
A: 删除 `server/data/jade-daily.db` 文件，重启服务后会自动重建数据库。

**Q: 日历数据显示不完整？**
A: 确保待办事项的 `date` 字段格式为 YYYY-MM-DD。
