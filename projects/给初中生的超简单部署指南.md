## 📋 准备工作清单

在开始之前，你需要准备：
- ✅ 一台电脑（Windows 或 Mac 都可以）
- ✅ 一个能上网的浏览器
- ✅ 一个邮箱（QQ邮箱、163邮箱都可以）
- ⏰ 预计总时间：15-20分钟

## 🚀 第一步：注册 Vercel 账号（5分钟）

Vercel 是一个免费的网站托管平台，就像给你的APP找一个免费的家！

### 1.1 打开注册网站
在浏览器里打开这个网址：
```
https://vercel.com/signup
```

### 1.2 选择注册方式
你会看到几个选项：
- **Continue with GitHub** （用GitHub注册）
- **Continue with GitLab** （用GitLab注册）
- **Continue with Bitbucket** （用Bitbucket注册）

**推荐：选择 "Continue with GitHub"**

如果你没有 GitHub 账号：
1. 点击 GitHub 选项
2. 会跳转到 GitHub 注册页面
3. 点击 "Sign up"（注册）
4. 填写：
   - Username：随便起一个用户名（比如：jade123）
   - Email：你的邮箱
   - Password：设置一个密码
5. 点击 "Create account"
6. GitHub 会发一封邮件到你邮箱，打开邮件点击验证链接
7. 验证完成后，回到 Vercel 页面继续

### 1.3 完成 Vercel 注册
回到 Vercel 页面后：
1. GitHub 会问你是否授权 Vercel 访问，点击 **"Authorize Vercel"**
2. Vercel 会问你一些个人信息：
   - Full Name：你的名字（中文也可以）
   - Work Email：你的邮箱
3. 点击 **"Continue"**
4. 创建你的第一个项目：
   - Team Name：留空，直接点击 **"Skip"**
5. 🎉 恭喜！你已经注册成功了！

## 🎨 第三步：上传到 Vercel（3分钟）

有两种上传方式，选择你喜欢的！

### 方式A：上传压缩包（最简单）

**如果你成功下载了文件：**

1. 解压文件
   - Windows：右键点击文件 → "解压到当前文件夹"
   - Mac：双击文件自动解压
   - 解压后会得到一个 `dist-web` 文件夹

2. 打开 Vercel 网站
   - 登录 https://vercel.com/
   - 点击右上角 **"Add New..."**
   - 选择 **"Project"**

3. 上传文件
   - 在页面中间有一个 **"Browse"** 按钮
   - 点击它
   - 找到你解压出来的 `dist-web` 文件夹
   - 选中整个文件夹，点击 **"Upload"**

4. 等待上传完成
   - 上传可能需要1-2分钟
   - 上传完成后点击 **"Deploy"**（部署）

5. 等待部署完成
   - 会看到绿色的进度条
   - 大约需要1-2分钟
   - 完成后会显示一个网址，比如：
     ```
     https://jade-daily-abc123.vercel.app
     ```
   - 🎉 记下这个网址！

### 方式B：用 GitHub 上传（推荐给有GitHub账号的人）

**如果你有 GitHub 账号：**

1. 创建 GitHub 仓库
   - 登录 https://github.com/
   - 点击右上角 **"+"** → **"New repository"**
   - Repository name：输入 `jade-daily-web`
   - 选择 **"Public"**（公开）
   - 点击 **"Create repository"**

2. 上传文件
   - 在仓库页面，点击 **"uploading an existing file"**
   - 拖动 `dist-web` 文件夹里的所有文件到网页上
   - 在底部输入 `Initial commit`
   - 点击 **"Commit changes"**

3. 在 Vercel 导入
   - 回到 Vercel
   - 点击 **"Add New..."** → **"Project"**
   - 在 "Import Git Repository" 里找到你的 GitHub 仓库
   - 点击 **"Import"**
   - 点击 **"Deploy"**
   - 等待完成，记下网址

## 📱 第五步：在飞书里添加你的APP（5分钟）

现在把你的APP添加到飞书里！

### 5.1 打开飞书开放平台
在浏览器打开：
```
https://open.feishu.cn/app
```

### 5.2 创建应用
1. 点击 **"创建应用"**
2. 选择 **"网页应用"**
3. 填写信息：
   - 应用名称：Jade's Daily Planner
   - 应用描述：我的日常管理小助手
   - 应用图标：可以随便上传一个图标（或者用默认的）
4. 点击 **"创建"**

### 5.3 配置网页地址
1. 在应用页面，找到 **"网页应用"** 或 **"网页"** 选项卡
2. 配置以下信息：
   - **PC端访问地址**：输入你的 Vercel 网址
     ```
     https://你的APP名.vercel.app
     ```
   - **移动端访问地址**：同样输入 Vercel 网址
3. 点击 **"保存"** 或 **"更新"**

### 5.4 发布应用
1. 点击 **"发布"** 或 **"上线"**
2. 填写一些发布信息（可选）
3. 点击 **"确认发布"**

### 5.5 测试
1. 打开飞书APP
2. 在工作台找到你的应用 "Jade's Daily Planner"
3. 点击打开
4. 🎉 成功了！

## ⚠️ 重要提示

### 关于后端（API）

现在你的APP只有前端（界面），后端（数据存储）还在我的服务器上。

这意味着：
- ✅ 你能看到界面
- ✅ 你能点击按钮
- ❌ 但数据可能无法保存（因为后端没在你的电脑上）

### 临时解决方案

如果你想测试功能，你可以：

1. **使用演示数据**：我可以帮你在前端加入一些演示数据
2. **稍后配置后端**：等你有了服务器，再配置后端

### 长期解决方案

如果你真的要用这个APP：

1. **购买一个云服务器**（大约 100-300 元/年）
   - 阿里云、腾讯云都可以
   - 选择最便宜的配置就够了

2. **部署后端**（我可以帮你）
   - 把后端代码放到服务器上
   - 配好数据库

3. **配置环境变量**
   - 在 Vercel 里设置后端地址
